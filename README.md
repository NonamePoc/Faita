## 🏃‍♂️ Run Locally

Clone the project

```bash
  git clone https://github.com/NonamePoc/MessageService/tree/client-app
```

Go to the project directory

```bash
  cd Front-End-Service
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
